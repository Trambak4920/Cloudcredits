# 📈 Stock Price Prediction

## Overview
Forecasts stock prices using time-series ML techniques.

## Techniques
- Feature Engineering (lag, rolling stats)
- Random Forest
- Gradient Boosting
- SVR

## Features
- Time-series split (no leakage)
- Scaling
- Next-day prediction

## Output
- Actual vs Predicted
- Residual plots
- Feature importance

## Run
```bash
python 9_stock_price_prediction.py
```
