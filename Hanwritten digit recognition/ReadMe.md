# 🔢 Handwritten Digit Recognition

## Overview
Classifies handwritten digits (0–9) using ML models.

## Models Used
- SVM (RBF)
- MLP Neural Network
- Random Forest

## Dataset
- sklearn digits (8x8 images)

## Output
- Confusion matrix
- Accuracy comparison
- Prediction visualization

## Run
```bash
python 3_handwritten_digit_recognition.py
```
