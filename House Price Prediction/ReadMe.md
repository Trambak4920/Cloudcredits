# 🏠 House Price Prediction

## Overview
Predicts housing prices using regression models on the California Housing dataset.

## Models Used
- Linear Regression
- Random Forest
- Gradient Boosting

## Metrics
- MAE
- RMSE
- R² Score

## Key Features
- Feature scaling
- Model comparison
- Feature importance visualization

## Output
- Actual vs Predicted plots
- Residual distribution

## Run
```bash
python 1_house_price_prediction.py
```
